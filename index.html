Bem-vindo ao PulseProxy Framework!

O PulseProxy é um framework de API Gateway leve e eficiente, construído em Go, projetado para rotear e gerenciar o tráfego de requisições para seus serviços de backend.

Ele oferece funcionalidades essenciais para a construção de microsserviços e APIs robustas, garantindo segurança, observabilidade e flexibilidade.

Principais Características:

  • Roteamento Dinâmico: Configure suas rotas facilmente via arquivo YAML.
  • Hot-Reloading: Altere configurações e rotas em tempo real sem reiniciar a aplicação.
  • Segurança: Suporte a chaves de API e políticas CORS configuráveis.
  • Observabilidade: Sistema de logging integrado para monitoramento de requisições.
  • Extensibilidade: Arquitetura modular para fácil adição de novos middlewares e funcionalidades.

Este framework é ideal para desenvolvedores que buscam uma solução performática e customizável para gerenciar o acesso às suas APIs.
